# /triage-inbox

Triage a set of emails, messages, or notifications and produce a prioritised action plan.

## Steps

1. **Categorise each item**:
   - Urgent + important → respond today
   - Important, not urgent → schedule for this week
   - Urgent, not important → delegate or decline politely
   - Neither → archive

2. **For each item requiring a response**: draft the response or delegation note.
   - Responses should be direct, complete, and no longer than needed
   - If the principal needs to make a decision: present the options clearly, with a recommendation

3. **Surface the priority list**: ordered by urgency, with suggested response approach for each

## Output format

**[Sender] — [Subject] — [Urgency/Importance classification]**
Suggested action: [reply / delegate / archive / schedule]
Draft: [if a reply is needed]
